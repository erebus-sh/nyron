# Installation

Install Nyron as a development dependency in your project:

<Tabs items={['bun', 'npm', 'pnpm', 'yarn']}>
  <Tab value="bun">
    ```bash
    bun add -D @nyron/cli
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install -D @nyron/cli
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add -D @nyron/cli
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add -D @nyron/cli
    ```
  </Tab>
</Tabs>

## Next Steps

After installation, proceed to the [quickstart guide](/docs/getting-started/quickstart) to initialize Nyron in your project.
